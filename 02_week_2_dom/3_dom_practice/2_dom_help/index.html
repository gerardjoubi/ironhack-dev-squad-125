getAttribute vs setAttribute vs removeAttribute

If you need to access an element's html attribute, element.getAttribute() comes to the rescue.
To an element's attribute as needed, element.setAttribute() is a tool choice.
To remove a specific attribute from a element, use element.removeAttribute().

  • element.classList.remove()

    Remove one or more CSS class(es) from an element

  • element.classList.contains()

    Check if an element contains specified CSS class

Insert textNodes

Remember, any HTML document is a tree of nodes.
These nodes can be text OR tags.
document.createTextNode() is an efficient solution to insert a text node inside an element.

var newTextNode = document.createTextNode("your new text node");
titleMain = document.getElementById("title_main");
titleMain.appendChild(newTextNode);

You may recognize a similar pattern as the one we used to insert circles !

NodeLists/HTMLCollections to Array

NodeLists and HTMLCollections are Array-like data structures.
At some point, you may find convenient to convert them as regular arrays.
Below, two techniques to perform such task,
using the from() function, tied to Array.prototype or es6 spread operator

const links = document.querySelectorAll(".link");
const h2s = document.getElementsByTagName("h2");
console.log(Array.from(links));
console.log([...h2s]);
